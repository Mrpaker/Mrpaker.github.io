新增值

现在你拥有了一个空的对象。但是一个空对象是完全没用的。在我们增加一些属性之前，我们需要理解在JavaScrpit中什么是属性。

属性有一个名字和一个值。此外，属性可以是可枚举的(enumerable)，可配置(configurable)的和可写的(writable)。

如果值是可枚举的，那么当对象使用for (prop in obj)循环枚举时，这个属性就会被枚举出来；

如果属性是可写的，那么你可以使用别的值代替它；

如果属性是可配置的，你可以删除或者改变他的其他属性（enumerable, configurable, writable）。

通常，当我们创建一个属性，我们会希望他是可枚举的(enumerable)，可配置(configurable)的和可写的(writable)。that was the only kind of property a user could create directly.

原型

到目前为止，我们已经介绍了对象就是简单的键值对。事实上，JavaScript对象还有一个额外的属性：一个指向其他对象的指针。我们管这个指针叫做对象的原型。如果你尝试去查找一个对象的key，但是没有找到，JavaScript将会在他的原型中查找。它会遵循”原型链”，直到它找到一个null的值。事实上，它会返回undefined。

你还记得我们通过Object.create(null)来创建一个新对象吗？这个方法传的参数，告诉了JavaScript，应该设置什么值给对象的原型。你可以查找一个对象的原型通过Object.getPrototypeOf:
